# Fashion MNIST Image Classifier (Streamlit)

## Опис роботи

У даній роботі було створено веб-застосунок для візуалізації роботи нейронних мереж, навчених на датасеті **Fashion MNIST**.  
Застосунок реалізований за допомогою бібліотеки **Streamlit**, яка дозволяє створювати прості веб-інтерфейси для демонстрації моделей машинного навчання.

Метою роботи було продовження домашнього завдання з модуля **"Згорткові нейронні мережі"** та створення веб-інтерфейсу для взаємодії з навченою моделлю.

У створеному застосунку користувач може:
- обрати модель для класифікації
- завантажити зображення
- переглянути результат класифікації
- побачити ймовірності для кожного класу
- переглянути графіки навчання моделі

Таким чином веб-застосунок дозволяє інтерактивно продемонструвати роботу нейронної мережі.

## Використаний датасет

Для навчання моделей використано датасет **Fashion MNIST**.

Fashion MNIST містить зображення предметів одягу у форматі grayscale розміром **28×28 пікселів**.  
Датасет складається з **10 класів**:

- T-shirt/top
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle boot

## Побудова моделі CNN

Першою моделлю у роботі є **згорткова нейронна мережа (CNN)**.

Архітектура моделі включає:

- згорткові шари Conv2D
- шари нормалізації BatchNormalization
- шари підвибірки MaxPooling
- повнозв'язні Dense шари
- Dropout для зменшення перенавчання

Модель була навчена на тренувальному наборі Fashion MNIST.

Після навчання модель була збережена у файл:

model_cnn.h5

Також було збережено історію навчання:

history_cnn.pkl

Це дозволяє відображати графіки функції втрат та точності.

## Використання моделі VGG16

Другою моделлю у роботі є **VGG16**, попередньо навчена на датасеті ImageNet.

Для адаптації моделі до задачі класифікації одягу було використано підхід **transfer learning**.

Було виконано такі кроки:

- завантажено базову модель VGG16
- заморожено ваги базової моделі
- додано нові Dense шари для класифікації

Історія навчання цієї моделі була збережена у файл:

history_vgg16.pkl

## Створення веб-застосунку

Для демонстрації роботи моделей було створено веб-застосунок за допомогою **Streamlit**.

Основні можливості застосунку:

- вибір між двома моделями (**CNN або VGG16**)
- завантаження зображення користувачем
- відображення завантаженого зображення
- виконання класифікації
- відображення передбаченого класу
- показ ймовірностей для кожного класу
- побудова графіків навчання моделі

Таким чином користувач може інтерактивно перевірити роботу нейронної мережі.

## Структура проєкту

fashion_streamlit_app

app.py – основний файл веб-застосунку  
utils.py – допоміжні функції для обробки зображень і передбачення  
README.md – опис проєкту  
requirements.txt – список необхідних бібліотек  

model_cnn.h5 – навчена CNN модель  
history_cnn.pkl – історія навчання CNN  
history_vgg16.pkl – історія навчання VGG16  

## Як запустити застосунок

### 1. Встановити необхідні бібліотеки

У терміналі потрібно виконати:

pip install streamlit tensorflow numpy matplotlib pillow

або

pip install -r requirements.txt

### 2. Запустити веб-застосунок

У папці проєкту виконати команду:

streamlit run app.py

### 3. Відкрити застосунок у браузері

Після запуску Streamlit відкриється сторінка:

http://localhost:8501

## Як користуватися застосунком

1. Вибрати модель у списку **Choose model**:
   - CNN
   - VGG16

2. Переглянути графіки навчання моделі:
   - Loss
   - Accuracy

3. Натиснути кнопку **Upload clothing image**.

4. Завантажити зображення.

5. Після завантаження застосунок:
   - відобразить зображення
   - покаже передбачений клас
   - покаже ймовірності для кожного класу
   - побудує графік ймовірностей.

## Висновок

У результаті виконання роботи було створено веб-застосунок для демонстрації роботи нейронних мереж для класифікації зображень одягу з датасету Fashion MNIST.
Застосунок дозволяє зручно перевіряти результати роботи моделей та візуалізувати процес їх навчання.
